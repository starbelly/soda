============

Tasty and refreshing libsodium bindings for Erlang and Elixir

## Installation

# Rebar

```erlang
{deps, [
    {soda, "0.1.0"}
]}
```

# Mix 

```elixir
def deps do
  [
    {:soda, "~> 0.1.0"}
  ]
end
```
